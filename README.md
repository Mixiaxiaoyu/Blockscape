# 方境 Blockscape

把现实带进方块世界。  
Bring reality into a world of blocks.

上传一张图片，方境会直接把它编辑成高质量方块游戏画面，并尽量保留原图的主要构图、姿势、道具和光照关系。

## 基本用法

```text
跑一下这张
把这张图变成 Minecraft 风格
Minecraft 化这张照片
```

默认会把人物和环境一起方块化，严格保持上传原图的像素尺寸和宽高比，不自动添加 UI 或准星，并优先一次完成。

## 快捷指令

### `@真`

快速保留照片中的真人，只把环境和背景改造成方块世界。

- 直接编辑上传原图；
- 不描述或重新生成人物；
- 默认只生成一次；
- 第一张没有明显人物漂移时立即返回；
- 后续添加 UI、准星或改比例时，直接编辑已接受结果。

```text
跑一下这张 @真
```

### `@UI`

输出为 16:9，并添加一套生存游戏 HUD：

- 九格快捷栏和选中格；
- 红心生命值、饥饿值和经验条；
- 7–9 个清晰且不同的像素道具。

`@UI` 默认不添加准星。

```text
跑一下这张 @UI
```

### `@准星`

输出为 16:9，添加完整生存 HUD，并额外添加一个居中的像素准星。`@准星` 自动包含 `@UI`。

```text
给这张图加上 @准星
```

### `@锁`

启用严格像素级人物锁定。

仅适用于用户已经提供可靠人物蒙版，或当前工具具有可靠原生区域蒙版的情况。该模式会进行原始人物像素回贴和验证，速度比普通 `@真` 更慢。

普通用户只需要使用 `@真`。

## 常用组合

```text
跑一下这张 @真 @UI
```

在一次编辑中保留真人、转换环境、扩展为 16:9 并添加完整 HUD，不添加准星。

```text
跑一下这张 @真 @UI @准星
```

在完整 HUD 上添加一个居中准星；不会重复添加 HUD。

## 模式区别

- `@真` = 快速原图编辑，默认一次生成。
- `@锁` = 严格像素合成，高级模式，需要可靠蒙版。
- 没有 `@UI` 或 `@准星` = 保持上传原图的像素尺寸和宽高比。
- `@UI` = 16:9 和完整生存 HUD，不含准星。
- `@准星` = 16:9、完整生存 HUD和一个居中准星。
